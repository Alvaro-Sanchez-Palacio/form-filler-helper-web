Security & Privacy

How Form Filler Helper protects your data and privacy

Local Processing

All form analysis and data generation happens locally in your browser. No data is sent to external servers or third-party services.

  • Form field detection runs in browser
  • Data generation uses local algorithms
  • CSV processing handled client-side

Minimal Permissions

Only requests the minimum permissions necessary for form filling functionality. No access to browsing history, bookmarks, or personal data.

  • Active tab access for form interaction
  • Storage permission for user preferences
  • No network or external access

Open Source

Fully open source codebase allows complete transparency. You can review every line of code to verify security and privacy practices.

  • Source code publicly available
  • Community-reviewed security
  • No hidden functionality

Data Isolation

Extension operates in isolated sandbox environment. Generated test data never mixes with your real personal information.

  • Sandboxed content script execution
  • Separate storage namespace
  • No cross-site data leakage
